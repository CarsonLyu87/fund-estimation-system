🎯 项目简介

这是一个基于Python的基金实时涨跌幅估算系统，专门为投资者提供准确的基金表现估算。系统使用天天基金网的持仓数据和东方财富网的股票实时数据，通过科学的算法计算基金的实时涨跌幅。

基金估算涨跌 = ∑(持仓权重 × 股票实时涨跌) + 现金收益

📈 实时估算

基于持仓和股票实时价格估算基金涨跌，提供准确的盘中表现预测。

🔢 多基金支持

可同时监控多只基金，支持自定义基金列表和优先级设置。

📊 智能报告

生成详细的估算报告，包含数据质量评估和投资建议。

⚡ 自动化运行

支持定时任务和自动化部署，无需人工干预。

🛡️ 错误处理

完善的异常处理和备用数据源，确保系统稳定运行。

🌐 开源免费

完全开源，代码透明，社区驱动，持续更新。

🚀 快速开始

只需几步即可开始使用基金估算系统：

# 1. 克隆仓库
git clone https://github.com/CarsonLyu87/fund-estimation-system.git

# 2. 进入项目目录
cd fund-estimation-system

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行估算
python scripts/run_estimation.py

# 5. 查看报告
cat reports/latest_fund_estimation.txt
