# 基金监控设置

## 概述
每日11点自动检查指定基金的涨跌情况，并通过微信通知。

## 已配置的基金
1. 华夏成长混合 (000001)
2. 易方达消费行业 (110022)
3. 招商中证白酒指数 (161725)
4. 南方纳斯达克100指数 (021000) - 从图片识别
5. 易方达蓝筹精选混合 (005827) - 从图片识别
6. 广发全球精选股票(QDII) (270023) - 从图片识别
7. 中欧医疗创新股票A (006228) - 从图片识别

## 文件说明
- `simple_fund_check.py` - 主监控脚本
- `fund_config.json` - 基金配置（可编辑）
- `fund_logs/` - 日志目录
- `FUND_MONITOR_SETUP.md` - 本文档

## 使用方法

### 1. 修改基金列表
编辑 `fund_config.json` 文件，更新 `funds` 数组：
```json
{
  "funds": [
    {"code": "基金代码", "name": "基金名称"},
    ...
  ]
}
```

### 2. 手动运行测试
```bash
python3 simple_fund_check.py
```

### 3. 查看日志
```bash
cat fund_logs/$(date +%Y%m%d).txt
```

### 4. 管理Cron任务
```bash
# 查看任务
openclaw cron list

# 删除任务
openclaw cron rm <任务ID>

# 添加新任务
openclaw cron add --name "任务名称" --cron "0 11 * * *" --message "运行基金检查脚本" --announce
```

## Cron任务详情
- 名称: daily-fund-check
- 时间: 每天11:00 (北京时间)
- 任务ID: 6d079409-2d76-406b-92f1-eba942cc0530
- 状态: 已启用

## 数据源
使用天天基金网API获取实时数据：
- 单位净值
- 估算净值
- 涨跌幅
- 更新时间

## OCR功能
系统支持通过OCR识别图片中的基金信息：
1. 发送基金截图
2. 自动识别基金代码和名称
3. 自动添加到监控列表
4. 标记来源为图片识别

## 注意事项
1. 基金数据在交易日15:00后更新
2. 非交易日可能无法获取最新数据
3. API可能有访问频率限制
4. 如需更多基金，请更新配置文件
